Fundamentos Essenciais

A base para você construir seu conhecimento.

O que é o Word?

Um editor de textos da Microsoft para criar, editar e formatar documentos. A extensão padrão dos arquivos é .docx.

A Interface

  • Faixa de Opções: Área principal de comandos.
  • Guias: Agrupam funcionalidades (Página Inicial, Inserir).
  • Grupos: Subdivisões das guias (Fonte, Parágrafo).
  • Atalho: CTRL+F1 oculta/mostra.

Gerenciamento de Arquivos

Use a guia Arquivo (modo Backstage) para ações como Salvar, Abrir (CTRL+O), Imprimir (CTRL+P) e Salvar Como (F12).

Edição e Formatação

Comandos para manipular e estilizar seu texto.

Manipulação de Texto

Copiar (CTRL+C): Duplica o texto selecionado.
Recortar (CTRL+X): Move o texto selecionado.
Colar (CTRL+V): Insere o texto da área de transferência.
Pincel de Formatação (CTRL+SHIFT+C/V): Copia e cola apenas a formatação.

Formatação de Fonte

Negrito: CTRL+N
Itálico: CTRL+I
Sublinhado: CTRL+S
Limpar Formatação: CTRL+ESPAÇO

Alinhamento de Parágrafo

Esquerda

Centralizar
CTRL+E
Direita
CTRL+G
Justificar
CTRL+J

Recursos Avançados

Funcionalidades poderosas para documentos complexos.

Guia Inserir

Adicione tabelas, imagens, gráficos, cabeçalhos, rodapés e quebras de página (CTRL+ENTER).

Guia Layout

Configure margens, orientação (retrato/paisagem), tamanho do papel e divida o texto em colunas.

Guia Revisão

Verifique a ortografia (F7), use o dicionário de sinônimos (SHIFT+F7) e conte as palavras do seu texto.

Tabelas

Ao criar uma tabela, novas guias contextuais ("Design da Tabela" e "Layout") aparecem para formatação avançada.

Estilos

Conjuntos de formatação pré-definidos que garantem consistência visual. Modificar um estilo atualiza todo o documento.

Localizar e Substituir

Encontre textos com CTRL+L e substitua com CTRL+U (no doc. original).

Guia Rápido de Atalhos

Use a busca para encontrar o atalho que precisa em segundos.
